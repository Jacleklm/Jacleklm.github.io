用 node.js 开发一个可交互的命令行应用

原生 node

  • 用 process.argv 直接读取
  • process.stdin.on 劫持输入
  • readline 模块 进行逐行读取。该模块提供了一个接口，用于一次一行地读取可读流（例如 process.stdin）中的数据

借助 prompt 包

如果你 打算在Node.js中构建可靠的CLI工具，则 prompt 可能是一个很好的选择。比readline更好用。
能像处理页面表单一样处理命令行输入
